Matchbox Car Replication - 3D Printing

Samantha Doniger

This is the third project we have completed for Engineering and Design 1. The goal of this project was to create a 3D model and 3D print of a toy car to scale.

Guidelines:

These are the guidelines for this project.

  • Car designed as an intersection of three forms
  • Design car in a 1:1 ratio
  • Size accuracy +/- 1%
  • Design is exported as STL
  • Student orients model in Cura for 3D printing to minimize support material
  • File is exported as a GCODE
  • Final design is shared out as an embedded STL on an HTML webpage

Project Steps:

These are the steps for creating the 3D car model.

  1. Draw
    • Draw the matchbox car to scale as an Engineering Drawing
    • Scan or photograph drawing
    • Capture screenshots of each plane side
  2. 3D Modeling
    • Draw each profile on each side plane
    • Using the extrusion tool, remove the negative spaces from the 3D form
    • Use extrusion and fillet tools for finish details
    • Export STL file
  3. Slicing and Printing
    • Import design into Cura
      • Make any final scaling changes
      • Position 3D model on build plate to minimize support material
      • Setting to 0.6mm nozzle
    • Export as a GCODE file
    • Print model
  4. Documentation
    • CAD drawing with margin of error analysis
    • Import design into Modelo to make a 3D model
    • Reflection will be a HTML document with blurb and 3D render
    • Add photo of car next to model into Slide Deck

Setbacks and Solutions:

These are the setbacks I encountered while trying to create my car and the solutions I came up with to overcome them.

One of the setbacks I encountered was that when I tried to extrude the right wheels of the car forward and backward so that the wheels on both sides would stick out, I could not make them extrude on the both sides using the same wheel drawing because it made the wheel continous across the entire bottom of the car. In order to fix this, I created two drawings on either side of the car and then extruded them both out away from the car.

Process and Photos:

This is the description of the process I took for creating my car and images of the steps that I took.

I first created a Engineering Drawing of my car using exact measurements that I calculated using calipers. The drawing contained three images of the car, one from the front, one from the side, and one from the top. In OnShape, I created a rectangle that had the same length, width, and height as the car. I took each view from my Engineering Drawing and pasted it onto its coordinating side of the car. I then removed the negative space to finalize the basic model of my car. I wanted to add a lot of details, so I created doors, side-view mirrors, a truck bed, front bumper details, wheel designs, and seats. I made the interior of the car hollow so that you could see through the windows and the front windshield into the seats. I then created an STL file and a GCODE file that I uploaded to the 3D Printer to fabricate my design. When calculating my margin of error, I found that the size ratio of my car to the original car is 1:1.
